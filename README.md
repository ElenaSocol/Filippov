# Random Quote Generator

**Автор:** [Филиппов Пётр]

## Описание
Программа на Python с графическим интерфейсом, которая выдает случайные вдохновляющие цитаты. Каждая сгенерированная цитата сохраняется в историю.

## Функционал
- Генерация случайной цитаты из встроенного списка.
- Ведение истории просмотров.
- Поиск по истории (фильтрация по автору и тематике).
- Автоматическое сохранение истории в JSON.

## Инструкция по запуску
1. Убедитесь, что установлен Python 3.x.
2. Скачайте `quote_generator.py`.
3. Запустите файл командой: `python quote_generator.py`.

## Примеры использования
- Нажмите "Сгенерировать", чтобы увидеть новую фразу.
- Введите "Эйнштейн" в поле фильтра по автору, чтобы найти его цитаты в вашей истории.
- Проверьте файл `history.json` в папке проекта — там хранятся все ваши данные.
